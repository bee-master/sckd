	Простейший драйвер светодиодных индикаторов
	для SDK-1.1	v.1.0.0	 13-06-2007
	-------------------------------------------------------


				1. Назначение 

Программа разработана в учебных целях для использования совместно с 
учебным стендом SDK-1.1.


  				2. Требования к платформе разработки


Сборку тестов можно производить в среде Linux или Win32/cygwin. 

Для сборки проекта вам понадобятся:
	- компилятор SDCC v2.7.0 или старше для вашей платформы; 
	- инструментальная программа (G)M3P.

Для архивации проекта вам понадобятся утилиты tar и gzip.

				3. Файлы проекта

makefile	- командный файл для утилиты GNU make
load.m3p	- командный файл для утилиты M3P/GM3P	
В обоих файлах в качестве com-порта прописан com1. Если используется 
другой com-порт, нужно отредактировать эти файлы.

SRC 		- исходные тексты программ
INCLUDE		- заголовочные файлы

				4. Компиляция и загрузка

Для сборки системы используйте утилиту GNU make. 

make 		- сборка проекта
make clean  - очистка 
make load	- загрузка проекта в учебный стенд
make term	- вызов эмулятора терминала
make dist   - подготовка архива tar.gz
